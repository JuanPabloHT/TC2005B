Mi videojuego favorito

Uno de los videojuegos que empecé a jugar hace un par de años, cuando recién armé mi propia computadora, tras meses de trabajo y ahorro, fue "Valorant", un videojuego perteneciente a la empresa Riot Games, creadora de otros videojuegos populares como "League of Legends". Valorant es un shooter que consiste en dos equipos de cinco personas, donde el equipo atacante tiene que plantar una bomba que lleva el nombre de "spike", mientras que el equipo defensor tiene que evitar que esta bomba sea plantada y, en caso de que ya haya sido plantada, desactivarla. Este juego comparte características de muchos otros. Desde mi perspectiva, es muy parecido en cuanto a dinámica a Counter-Strike, otro shooter muy popular, y a Overwatch, porque los personajes, en este caso "agentes", además de tener un arma como en cualquier shooter, tienen habilidades especiales y cumplen un rol. Dediqué horas y horas a este juego, principalmente en el modo competitivo, donde la principal característica es que juegas para subir de rango. Podríamos decir que las partidas se toman más en serio, además de que están igualadas. Desde el primer momento me fascinó la complejidad estratégica que tenía el juego, además de las mecánicas, que, si bien son parecidas a otros juegos, yo no conocía.

Importancia de los videojuegos en mi vida

Desde que tenía 7 años, los videojuegos han estado muy cerca de mí. Jugué en arcades y en consola (PSP), siempre disfruté mucho de ellos, ya fuera cuando viví momentos difíciles o incluso solo para pasar el rato. Si bien al inicio no eran más que un pasatiempo más, cuando crecí y empecé a jugar en línea, fueron el medio para conocer personas que compartían ese gusto conmigo. Fueron una herramienta para hacer amigos; puedo decir que a algunos de mis mejores amigos actuales los conocí de esa forma. Hoy tengo grandes recuerdos con ellos, jugando hasta tarde y platicando de mil cosas. Gracias a ellos, aprendí mucho fuera de los videojuegos. Los videojuegos, más allá de un simple pasatiempo, fueron el medio para conectar con otras personas.

Preguntas

¿Cuál es la diferencia entre Internet y la World Wide Web? La Web(WWW) es el servicio mas grande en internet, utilizado por la mayoria de personas, pero el internet es mas que solo la web,es una red informática que conecta, a escala mundial, miles de redes a diferentes niveles.
¿Cuáles son las partes de un URL? Las URL estan conformadas por 4 partes: Un esquema (protocolo que debe utilizarse para acceder al recurso en Internet), Un nombre de host (Identifica el host que contiene el recurso), un camino (La ruta identifica el recurso específico en el host al que el cliente web quiere acceder) y una cadena de consulta (Sigue al componente de ruta y proporciona una cadena de información que el recurso puede utilizar para algún fin).
¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE? Cada uno de los metodos cumplen funciones especifica, GET (Para recuperar datos, información + contenido, serán visibles en la URL), HEAD (De igual manera recupera datos, pero esta vez solo información, en especifico los encabezados, no el contenido), POST (Es usado para enviar datos, de los que el servidor hace uso a traves del cuerpo), PUT (De igual manera usado para enviar datos, con la diferencia de que estos datos reemplazan o actualizan los datos o representaciones presentes), PATCH (Similar al anterior, sirve para hacer actualizaciones a los datos presentes, con la diferencia que se realiza de forma parcial) y DELETE (Sirve para borrar un dato presente especifico).
¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué? El método POST porque vamos a enviar información, de esta forma esta segura porque no será visible mediante el url.
¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL? En este caso se utiliza GET, porque al acceder a una pagina recuperamos la información de la misma sin modificarla.
Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error? No, es una respuesta satisfactoria, en este caso indica que la solicitud fue exitosa. ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué? Tomando en cuenta que el error 404 indica que la pagina no ha sido encontrada, quien debe corregirlo es el desarrollador, dado que debe asegurarse de que los enlaces funcionen adecuadamente.
¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué? Este error indica un problema interno del servidor, considero que es responsabilidad del desarrollador arreglarlo, ya que es parte de sus responsabilidades asegurarse que este funciones correctamente.
¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados. Indica que el atributo ha quedado obsoleto y no se recomienda su uso. Algunos ejemplos son: acronym, bgsound, frame, etc.
¿Cuáles son las diferencias principales entre HTML 4 y HTML5? la compatibilidad con navegadores, las etiquetas usadas, soporte multimedia, formularios mejorados para html5 y las nuevas API's para este.
¿Qué componentes de estructura y estilo tiene una tabla? Leyendas, cuerpos, encabezados y pies de tablas, filas, columnas y celdas. ¿Cuáles son los principales controles de una forma HTML5? Entrada de texto, contraseña,contenido oculto, casillas de verificación, casillas de opción, botones, selector de archivos, entre otros.
¿Qué tanto soporte HTML5 tiene el navegador que utilizas? En mi caso uso brave, haciendo uso del test de la pagina: https://html5test.opensuse.org/, mi navegador puntuó 538 de 571.
Sobre el ciclo de vida y desarrollo de los sistemas de información: ¿Cuál es el ciclo de vida de los sistemas de información? Es un sistema, automatizado o manual, que engloba a personas, máquinas y/o métodos organizados para recopilar, procesar, transmitir datos que representan información.
¿Cuál es el ciclo de desarrollo de sistemas de información? Es un conjunto de fases que se llevan a cabo al desarrollar un sistema de información.
Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS? Respecto al uso de !important, es recomendable no hacer uso del mismo a menos que sea necesario.
Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado? Estas pueden afectar la lectura de textos, ademas si se cambian las dimensiones está se puede distorsionar empeorar la estetica de tu pagina, entre otras cosas.
Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt? la unidad % es útil para diseños responsivos, la px para medidas fijas y pt para asignar un tamaño a la fuente de un texto, donde resulta muy util pues es consistente en los diferentes dispositivos.
¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio? por que reduce el tamaño de los archivos de css, eliminando todos los elementos innecesarios.
